# Oriented-Items-for-Honeycomb-Cardboard-Production
# Data and Code for "Incorporating Item Orientation in the Two-Dimensional Variable-Sized Guillotine Cutting Stock Problem for Honeycomb Cardboard Production"

**Authors**  
Jung-Fa Tsai¹, Vu Ngoc Linh Le¹, Ming-Hua Lin²,*  

¹Department of Business Management, National Taipei University of Technology, Taipei, Taiwan  
²Department of Marketing and Management, University of Taipei, Taipei, Taiwan  
*Corresponding author: mhlin@utaipei.edu.tw

## Description
This repository contains the data and results of the MILP model proposed in this paper.  
The model extends the formulation of Terán-Viadero et al. (2024b) by introducing binary item-orientation variables.
